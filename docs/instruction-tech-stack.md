## 2. 기술 스택

- **프레임워크**: Next.js v16 (App Router)
- **언어**: TypeScript
- **스타일링**
  - shadcn v4 (UI 라이브러리)
  - Tailwind v4 (CSS 라이브러리)
- **상태 관리**
  - Zustand (전역 상태관리)
  - Tanstack Query (비동기 상태관리)
- **테스트**
  - Playwright (컴포넌트, E2E)
  - Vitest (단위)
- **기타 주요 패키지**
  - Tanstack Form (폼 제출 동작관리)
  - Tanstack Virtual (렌더링 최적화)
