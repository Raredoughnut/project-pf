Raredoughnut · Design System v0.1

A shared warm system
for rare products.

Foundations and components behind xpf (extra profile for X) and otogedojo (music-game skill todolist). Built on top of shadcn/ui + Tailwind tokens already used in both codebases — reskinned with a warm doughnut palette.

⚑
Substitutions flagged. Pretendard (Korean UI), Fraunces (display serif) and Geist Mono are used in place of an unspecified brand type system. Both source codebases ship with default shadcn/ui grayscale tokens — the warm palette below is derived from the Raredoughnut logo and is the new source of truth.

Brand & logo

The mark centers a "rare" glazed doughnut — cream interior, warm crust, orange sprinkle. Keep clearspace equal to the doughnut's hole on all sides.
